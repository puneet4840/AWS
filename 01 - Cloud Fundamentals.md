### What is cloud?

Cloud ka matlab hai data center ko internet se access karna.

Cloud ka matlab hai ki remote location par rakhe hue server, storage, networking ko use karna.

**Pehle Basics Samjho**

### Local System Kya Hota Hai?

Imagine karo:
- Tumhare paas ek laptop hai.
- Uske saath mouse, keyboard, monitor, external HDD, charger.
- Sab tere room mein hi connected hai.

Yahi hota hai:
- Local System ya Local Environment.

Isme Sab kuch tera hota hai, Hardware ka control tera, Fail ho jaye → tu repair karega, Storage bhar jaye → tu upgrade karega, Power chala jaye → system band ho jayega, Ek time par limited performance.

**Real Example**:

VS Code install, Docker local setup, local PostgreSQL, sab teri machine par.

<br>

### Datacenter Kya Hota Hai?

Ab soch ek doosri country mein ya tere hi city mein ek bahut bada hall hai.

Us hall ke andar:
- Bohot saare powerful computers hain.
- Gigabytes mein networking hai.
- Petabytes mein storage hai.
- Power supply hai.
- Cooling systems hain.

Yahi hota hai Datacenter.

AUr isi datacenter ko jab internet se access karte hain aur servers ko use karte hain, usi ko cloud bolte hain.

Cloud ka matlab hai ki remote location par rakhe hue servers ko use karna.

<br>

### Cloud Computing kya hai?

Cloud computing ka matlab hai ki servers, storage, networking, database, jesi services ko on-demand internet ke through access karna.

Humne uper datacenter padha kya hota hai, to companies jaise AWS, Microsoft, Google, etc, ne alag-alag jagah par apne datacenters bana rakhe hain. Hum jab chahe aur jitna chahe in companies ko paise dekar service use kar sakte hain. Isi concept ko cloud computing kehte hain.
