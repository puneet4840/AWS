### Local System Kya Hota Hai?

Imagine karo:
- Tumhare paas ek laptop hai.
- Uske saath mouse, keyboard, monitor, external HDD, charger.
- Sab tere room mein hi connected hai.

Yahi hota hai:
- Local System ya Local Environment.

Isme Sab kuch tera hota hai, Hardware ka control tera, Fail ho jaye → tu repair karega, Storage bhar jaye → tu upgrade karega, Power chala jaye → system band ho jayega, Ek time par limited performance.

**Real Example**:

VS Code install, Docker local setup, local PostgreSQL, sab teri machine par.
